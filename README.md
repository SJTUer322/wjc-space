# 王久畅的空间

这是一个简洁前卫风格的个人主页项目，支持：

- 展示个人主页标题
- 跳转至中国人民公安大学官网
- 跳转至徐州市第一中学官网（https://www.jsxzyz.cn/）
- 浏览器本地访问日志记录
- 移动端适配

## 文件说明

- `index.html`：主页结构
- `style.css`：页面样式
- `script.js`：交互与本地日志功能

## 部署方式

可直接上传到 GitHub 仓库，并使用 GitHub Pages 发布。

## 注意事项

当前日志功能为浏览器本地记录，仅能记录当前设备上的访问和点击行为，不是服务器端全站访问日志。
